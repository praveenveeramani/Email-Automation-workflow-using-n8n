# Email-Automation-workflow-using-n8n

This project automates Gmail responses using n8n. It classifies incoming emails into categories 
(Important, Personal, Transactional, Other) and auto-generates context-based reply drafts.

## Features
- Real-time Gmail trigger
- Automatic email classification
- Smart draft reply creation
- No-code AI automation using n8n

## How to Use
1. Import `Email automation.json` into your n8n instance.
2. Connect your Gmail and OpenAI credentials.
3. Activate the workflow to start auto-drafting replies.
